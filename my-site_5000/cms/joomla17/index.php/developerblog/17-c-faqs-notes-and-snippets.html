Q. Write a function that will return an aligned address ?

A.

 T* AlignPointer(void* raw, int align) // align = 0,1,2,4 ... 
{
	T* p = (raw + align - 1) & ~(align - 1);
	return p; 
}

Q. What is the range of Signed/Unsigned Char ?

A. A signed char with a hexadecimal value of 0xFF is equivalent to the decimal number -1. An unsigned char with a hexadecimal value of 0xFF should display as decimal 255.

Signed integers use the MSB (Most Significant Bit) as a sign bit where 0 indicates a positive value and 1 indicates a negative value. This notation is known as 2's complement notation.

Since the MSB is the sign bit, we are left with 7 bits to express magnitude. For positive values, the range is 0x00 (0d) to 0x7F (127d). For negative values, the range is 0x7F (1d) to 0x00 (128d). For an 8-bit value, the most negative, decimal -128 is 0x80, decimal 0 is 0x00 and the most positive value, decimal 127, is 0x7F.

To convert a decimal value to a signed binary representation, write its magnitude in binary. If the value is negative, take the 2's complement. The 2's complement consists of the 1's complement plus one. The 1's complement is taken by simply inverting each bit.
 
Decimal -1: 0 000 0001 - 0x01 (I separated the MSB for clarity.)
1's complement: 1 111 1110 - 0xFE
2's complement: 1 111 1111 - 0xFF
 
Decimal 25: 0 001 1001 - 0x19
Since this is positive, we're done.
 
Decimal -41: 0 010 1001 - 0x21
1's complement: 1 101 0110 - 0xD6
2's complement: 1 101 0111 - 0xD7
 
To convert signed binary values to decimal, you simply reverse the process:
Decimal -107 = 0x95
2's complement: 1 001 0101 - 0x95
1's complement: 1 001 0100 - 0x94
Decimal 107: 0 110 1011 - 0x6B
 
From the values you gave in your table, I'm guessing that your text was written for 16-bit integers. For 32-bit integers, your table becomes:
The decimal of signed char 0xFF is -1.
The decimal of unsigned char 0xFF is 255.
The decimal of signed int 0xFFFFFFFF is -1.
The decimal of unsigned int 0xFFFFFFFF is 4294967295.
The hex of decimal 12345 is 0x00003039.
The hex of decimal -12345 is 0xFFFFCFC7.
