My personal favorite Joomla 1.5 extension

Last Updated on Tuesday, 27 December 2011 20:50 | | Hits: 724

Joomla is one of the great CMS software that come with tons of features by default. However, if you want to have your website to be able to represent a specific feature, using the extensions is unavoiable. Here I listed a number of useful extensions for your Joomla 1.5 base website

Webpage Presentation
  • avreloaded - embedding internal/external video
  • random-image - a good small module, it will random pick the picture and show on your frontpage :) not that powerful but very easy to use.
  • CodeCitation - add ability to show C++/etc code syntax in the article
Content Creation
  • Code Mirror - my favorite editor, aimming for coder. it is a blank editor with html/php/etc syntax highlighter
  • JCE Editor - more powerful content editor, make Joomla article editor looks similar to MS word
  • YvBBCode - pluging that allow you to combine some BBcode into your article
  • CodeCitation - add ability to show C++/etc code syntax in the article
  • editor-switchers - add the ability to switch between any above editors in realtime. very handy to have 5/5 from me
