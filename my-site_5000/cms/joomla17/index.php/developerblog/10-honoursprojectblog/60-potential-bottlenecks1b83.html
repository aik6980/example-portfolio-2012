From some of the above documents, I can draw some of interesting fact about DirectX resources as following

  • It's more restrictive to use DirecX10 buffer compare to DirectX9. The usage flags will determine where is the buffer will generally stay (GPU or CPU). The access flags and buffer type will determine the buffer accessibility
  • some type of buffers can perform very fast memory operations (copy, move) on GPU through GPU command functions. However, they are nor accessible from CPU and it's very hard to modify the content
  • CPU accessible buffer is easier to manipulate but it's slower to perform. We can access this kind of buffer by using mapped()
