Development Note

  • DXGI (DirectX Graphic Infrastructure) is the new foundation class that is introduced for the first time in DirectX10. The classes will manage and store the information of low-level structures, (which is lower than ID3D10Device), at the adapters level
  • DXGI interface allow us to query the adapter information, including any custom-adapter (as we used to query NVPerfHUD instrumentation driver)
  • We used MSAA supported Backbuffers where the we need "Alpha-to-Coverage" rendering.
  • MSAA BackBuffer is almost, but not the same as any Normal BackBuffer. It has to be created with some specific flags
  • Please note that the RendertargetView Buffer and DepthStencilView Buffer, sizes and sample property need to be the same
