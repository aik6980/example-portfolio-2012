Developed in 2006, the application is part of my 3D programming practice using OpenGL API. Everything is rendered via the fixed-pipeline functionality.

The program is consisted of a few 3D scenes, which represent different techniques normally found in games. The experiments involved Basic Vertex and Pixel Lighting (Phong model), CPU Bone Animation, Particle System, Alpha Blending, Volumetric Fog and Etc.

NAME : OPENGL Application - 3D Scene
PLATFORM : WIN32
DEV : OPENGL 2.0, VC2005
CONTROL : Keyboard
