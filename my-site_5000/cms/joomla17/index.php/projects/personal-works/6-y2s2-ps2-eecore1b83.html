Y2S2 PS2 EECore

Last Updated on Monday, 09 January 2012 11:15 | Written by Super User | | Hits: 3518

SCEE Award Winner 2006

Developed on the Playstation 2, The project is created base on the famous DOOM game. Even idea is the same, but the game need totally different implementation to make it happens on PS2 Console.

The game obviously haven't used the real rendering of the console. It basically does all transformation and mapping work on the EECore, main CPU unit. Overall, it does give me a very good basic about the 3D rendering pipeline.

NAME : PS2 DOOM - Killing Field
PLATFORM : Sony Playstation 2
DEV : GCC in PS2 Linux kit
CONTROL : DUALSHOCK2 Joystick

Forward/Backward move Left analog stick, Direction Pad
Rotate L,R Button
Use weapon X Button
Quit Start + Select
