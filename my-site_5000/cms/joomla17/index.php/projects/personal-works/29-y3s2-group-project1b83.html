Project CEPA (2008)

Last Updated on Saturday, 07 January 2012 10:42 | Written by Super User | | Hits: 3141

Editor pick! : Project of the Year Award

CEPA Project is a full game project from a team of 5 people which I have taken part as a team leader. The project is produced as parts of my third year university’s Group project.

The main idea of the project is about the whole software development process, team organization, project management and software engineering. Involving as the team leader, I have to do all producer’s jobs, including schedule planning, documenting and motivating team members. I have learnt so many things during the project execution, not only programming and designing but also about managing and team mentoring where demonstrating professional behaviors and maintaining inter-personal relationships among members are very important.

Apart from the Team Leader, another role is also to take care all graphic programming, and implement most of core modules of the application.

 

Let's talk a bit about the game...

The main game idea is to bring the futuristic fast-pace multi-players, arcade style shooting game. Each player will control the robot with full heavy weapons, helping each others in order to destroy the unidentified alien that targeted in the center of the area. Apart from the Team Leader, I also take care most of the graphic programming, and part of the game-play programming implementation. The project use OGRE3D as the graphic API. Effects are created from Cg (NVidia Shading language)

Things I've learnt as a Team leader

  • Seriously mentoring and professional behaviour
  • Team motivating
  • Project planning, Time management

Things I've learnt as a Graphic programmer/Game programmer

  • OGRE3D, Core engine
  • Game-related foundation class, Resource manager, Object factory, State management and etc.
  • Developing Shaders for game elements, Explosions
  • Terrain generation, featuring Normal/Relief mapping
  • Animations
NAME : Project CEPA
PLATFORM : Windows XP
Dev Environment : Visual C++ 2005, OGRE3D, PhysX, FMOD
CONTROL : DUALSHOCK2 Joysticks
