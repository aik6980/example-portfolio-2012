DirectX HLSL (2007)

Last Updated on Saturday, 07 January 2012 10:25 | Written by Super User | | Hits: 3252

The program is developed using DirectX9 API. The main idea is to experiment and integrate the HLSL Shaders into my framework. The fixed-pipeline function also has been replaced with the equivalent shaders.

Shader programming becomes more and more popular, even is the industry standard now. It is no question that GPU has become more and more powerful nowadays, so learning how to use them properly will be a must to any Graphic programmer.

Since I am very interested in, and want to end up to be the graphic programmer in the industry. This is one of my most important project that I have a chance to explore many different techniques in GPU programming.

  • Water rendering using Displacement and Normals mapping
  • Random terrain generation with Textures splatting
  • Software/Hardware skinning animation included basic instancing
  • Real-time Reflections using dynamic cube-mapping
  • Simple Post-processing eg. Radial Blur, Blur effects
  • Shadow mapping
