Space Duel, This is my first game which is developed under Playstation2 console environment in 2006. Game Graphic is built on NCURSE, text-console extension library.

 Two players VS fighting style is the main game idea. In gameplay, players're allowed to upgrade their ship by select and picking up items. Game will finish when either player lose all lifepoint.

NAME : Space Duel - 2 Players VS Fighting
PLATFORM : Sony Playstation 2
DEV : GCC in PS2 Linux kit
CONTROL : DUALSHOCK2 Joysticks

Movement control Left analog stick, Direction Pad
Use weapon X Button
GUN Upgrade L Button
BULLET Upgrade R Button
Quit Start + Select

It is required PS2 Linux kit in order to play the game. However, I've provided gameplay video in case you want to see it in action!!
Special thanks to Andy "The Master" who help me test and beat me every time we play!
This game is developed by C++ under PS2 Linux kit. The game object is managed using STL library. Data structures is better constructed to handle more game functionality. The DualShock2 Controller features have been used and explored, include the analog pressure buttons and actuators control.
