Star Surfer, The game is developed in 2005 on a basic console using C language. I make this game in order to explore some basic data structures. The game simply shows how to build a simple game in a very straight forward way. The graphic output is the Win32 console, using the Andre Lamothe extended library. High score record is provided in order to study file I/O functions.

NAME : Star Surfer
PLATFORM : WIN32 Console
DEV : VC6.0
CONTROL : Keyboard 

  Move Left,Right A,Z
  Repair ship R

Welcome to Space surfing club! Your target is simple, try to stay int meteorites storm as long as possible. We know it's pretty hard to stay clean all the time, so after three hits your ship might be broken. However, you're allowed to buy the repair kit that cost is depended on the stage difficulty.
Hope to see you on the billboard.
