Based on sprites from Dragonball Z and Gundam Wing. This is my 2D Fighting game developed in 2006 for Windows, rendering using GDI. The game idea came from my favorite action manga written by Toriyama Akira. The story is about an adventure of a young boy names Goku, and his friends who trying to find set of the magical crytals that can give you anything you want just in one pray.

NAME : Dragonball Z
PLATFORM : WIN32
DEV : GDI, VC2005
CONTROL : Keyboard

  Veggito Cell
Move Forward Backward D,G LEFT, RIGHT Arrow
Kick A NUMPAD -
Special power Z NUMPAD +
Jump R UP Arrow

The game scene is about the fight between Goku and Cell, their fighting scene is one of the most greatest fight ever which you can find through the story.
