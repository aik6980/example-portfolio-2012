Y1S2 GBA Alien

Last Updated on Monday, 09 January 2012 11:10 | Written by Super User | | Hits: 2606

Alien, a simple platformer developed on Game boy advance. I've got the idea from Super Mario, the classical platform game from Nintendo. The game contain one level with various atmosphere, from Castle to Church in Goth.


The game is written in C++ using DevkitADV sdk, an open-source toolkit based on GNU GCC. Another main purpose is to study the ARM CPU structure and GBA hardware features.

NAME : Alien Escape
PLATFORM : Nintendo Gameboy advance
DEV : GCC via Emulator
CONTROL : Keyboard/Joystick

setup via emulator

The game can be played using various GBA Emulator, which could be found quite easily over the internet. VirtualBoyAdvance(VBA) and MappyVM are my recommendation.
