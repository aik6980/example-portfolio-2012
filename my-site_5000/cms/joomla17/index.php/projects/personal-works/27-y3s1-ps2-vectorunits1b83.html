Y3S1 PS2 VectorUnits

Last Updated on Saturday, 07 January 2012 09:44 | Written by Super User | | Hits: 2958

The program has been developed in 2007, in order to explore and experiment the real power of PS2 3D pipeline. Using the Vector Unit 1/2 to perform the vertex operations.

 

The demo shows a few different VU program that handle different effects that could be able to implement into 3D game. We can see the program that provide the animation to our 3D character. Multiple lights that shade the scene, Skybox and Terrain/Detail implementation using Heightfield.

VUs are the special CPU (GPU) that decided to be able to handle very fast vertex operation, and that is the key to 3D rendering techniques on the PS2. Each VU provides little memory that user can create the custom program to produce the graphical effects.
 

 

NAME : PS2 3D GraphicPipeline
PLATFORM : Sony PlayStation 2
Dev Environment : PS2 LinuxKit
CONTROL : DUALSHOCK2 Joysticks
