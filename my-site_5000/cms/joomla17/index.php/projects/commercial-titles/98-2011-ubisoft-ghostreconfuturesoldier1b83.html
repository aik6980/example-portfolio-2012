Ghost Recon: Future Soldier. I joined the team to help optimizing and polishing the PS3 version of the game.

Roles and responsibilities

- Work remotely in collaboration with the rendering team in Paris, in order to improve the shader asset pipeline performance and offline optimization.

This project give me an opportunity to learn and work with an another good game engine. The architecture is very different yet very interesting comparing to my previous game, Driver:SF. Moreover, working on the different genre I can see a number of different design principles that allow the gameplay to have the best benefit.

Working remotely help improving my collaboration skills, programming techniques and etc. It is very nice to be able to learn to work and discuss among number of great, experienced programmers from the different studios.
